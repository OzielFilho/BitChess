# BitChess

## Visão Geral

O BitChess é um projeto com a finalidade de desenvolver uma experiência de xadrez clássico na plataforma Unity, enriquecendo-a com uma estética de pixel art. Este jogo abrangerá uma variedade de modos de jogo, incluindo partidas de jogador único contra uma IA, bem como partidas multijogador online. Além disso, estamos comprometidos em tornar o BitChess acessível a um público mais amplo, planejando lançá-lo em dispositivos móveis

## Objetivos

- Desenvolvimento do jogo de xadrez em estilo bit (pixel art)

- Inteligência artificial para jogo singleplayer

- Busca por partidas multiplayer

- Disponibilização para plataformas móveis
